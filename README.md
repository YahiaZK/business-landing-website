# Business Landing Website

A responsive business landing website.

## Project Overview

This project is a multi-page business website built with HTML and CSS as part of the Udacity Front End Web Developer Nanodegree Program.

## Features

- **Responsive Design** - Adapts to different screen sizes (mobile, tablet and desktop)
- **Home Page** - Hero section, company values cards, and about us section
- **Contact Page** - Contact form and company contact details
- **Clean Navigation** - Easy navigation between pages
